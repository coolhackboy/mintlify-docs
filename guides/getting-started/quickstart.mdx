## Welcome to PoYo AI

PoYo AI provides powerful image and video generation APIs that enable you to integrate state-of-the-art AI models into your applications. This guide will help you get started in just a few minutes.

## Authentication

### Get Your API Key

1. Visit the [PoYo AI Console](https://poyo.ai/dashboard/api-key)
2. Sign in or create an account
3. Navigate to the API Key Management page
4. Generate a new API key
5. **Important**: Copy and securely store your API key - it will only be displayed once

### Using Your API Key

Include your API key in the `Authorization` header of all API requests:

```bash
Authorization: Bearer YOUR_API_KEY
```

## Quick Examples

### Generate an Image

Here's how to generate an image using the GPT-4o Image model:

```python
import requests

url = "https://api.poyo.ai/api/generate/submit"
headers = {
    "Authorization": "Bearer YOUR_API_KEY",
    "Content-Type": "application/json"
}

payload = {
    "model": "gpt-4o-image",
    "callback_url": "https://your-domain.com/callback",
    "input": {
        "prompt": "A serene mountain landscape at sunset with vibrant colors",
        "size": "1:1",
        "n": 1
    }
}

response = requests.post(url, json=payload, headers=headers)
result = response.json()
task_id = result["data"]["task_id"]
print(f"Task ID: {task_id}")
```

```javascript
const response = await fetch('https://api.poyo.ai/api/generate/submit', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    model: 'gpt-4o-image',
    callback_url: 'https://your-domain.com/callback',
    input: {
      prompt: 'A serene mountain landscape at sunset with vibrant colors',
      size: '1:1',
      n: 1
    }
  })
});

const result = await response.json();
const taskId = result.data.task_id;
console.log(`Task ID: ${taskId}`);
```

```bash
curl -X POST https://api.poyo.ai/api/generate/submit \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "gpt-4o-image",
    "callback_url": "https://your-domain.com/callback",
    "input": {
      "prompt": "A serene mountain landscape at sunset with vibrant colors",
      "size": "1:1",
      "n": 1
    }
  }'
```

## Asynchronous Processing

All generation requests in PoYo AI are processed asynchronously. This means:

1. **Submit Request**: When you submit a generation request, you receive a `task_id` immediately
2. **Check Status**: Use the task ID to query the generation status
3. **Retrieve Results**: Once complete, the status endpoint returns the generated content

### Check Task Status

```python
import requests
import time

task_id = "task-unified-1757165031-uyujaw3d"
url = f"https://api.poyo.ai/api/generate/status/{task_id}"
headers = {
    "Authorization": "Bearer YOUR_API_KEY"
}

# Poll until complete
while True:
    response = requests.get(url, headers=headers)
    result = response.json()
    task = result["data"]

    status = task['status']
    progress = task.get('progress', 0)

    print(f"Status: {status}, Progress: {progress}%")

    if status == 'finished':
        print(f"Files: {task['files']}")
        break
    elif status == 'failed':
        print(f"Error: {task['error_message']}")
        break

    time.sleep(2)  # Wait 2 seconds before next check
```

```javascript
const taskId = 'task-unified-1757165031-uyujaw3d';

async function checkStatus() {
  while (true) {
    const response = await fetch(
      `https://api.poyo.ai/api/generate/status/${taskId}`,
      {
        headers: {
          'Authorization': 'Bearer YOUR_API_KEY'
        }
      }
    );

    const result = await response.json();
    const task = result.data;
    const { status, progress, files, error_message } = task;

    console.log(`Status: ${status}, Progress: ${progress}%`);

    if (status === 'finished') {
      console.log('Files:', files);
      break;
    } else if (status === 'failed') {
      console.log('Error:', error_message);
      break;
    }

    await new Promise(resolve => setTimeout(resolve, 2000));
  }
}

checkStatus();
```

```bash
curl -X GET https://api.poyo.ai/api/generate/status/task-unified-1757165031-uyujaw3d \
  -H "Authorization: Bearer YOUR_API_KEY"
```
