## Quick Example

Generate an image in just a few lines of code:

```python
import requests

response = requests.post(
    "https://api.poyo.ai/api/generate/submit",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
    json={
        "model": "gpt-4o-image",
        "prompt": "A serene mountain landscape at sunset",
        "size": "1:1"
    }
)

task_id = response.json()["id"]
```

## Key Features

### Asynchronous Processing

All generation requests are processed asynchronously for optimal performance:

1. Submit your request and receive a task ID instantly
2. Poll the status endpoint to check progress
3. Retrieve results when complete
4. Optional webhook callbacks for automatic notifications

### Unified API Design

Single endpoint for all models:

- **Submit**: `POST /api/generate/submit`
- **Status**: `GET /api/generate/status/{task_id}`

Switch between models by simply changing the `model` parameter.

### Flexible Parameters

Customize your generations with extensive parameters:

- **Images**: Size, quantity, reference images, masks
- **Videos**: Duration, aspect ratio, thumbnails, watermarks
- **Both**: Callbacks, custom parameters per model
