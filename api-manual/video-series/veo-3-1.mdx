# VEO 3.1 Video Generation

Generate high-quality videos quickly using Google's VEO 3.1 model. Optimized for speed while maintaining professional quality.

## Available Model

- **veo3.1-fast** - Fast 8-second video generation

## Key Features

- **Fixed Duration**: 8 seconds
- **Fast Generation**: ~90 seconds (2x faster than Sora 2)
- **Aspect Ratios**: 16:9 (landscape) and 9:16 (portrait)
- **Modes**: Text-to-video and image-to-video
