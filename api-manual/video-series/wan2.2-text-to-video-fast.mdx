# Wan 2.2 Text-to-Video Fast

`wan2.2-text-to-video-fast` is optimized for fast text-to-video generation with flexible aspect ratios and lightweight resolution options.

## Available Model

* **wan2.2-text-to-video-fast** - Generate short videos from text prompts

## Key Parameters

* **Aspect Ratio**: `16:9`, `1:1`, `9:16`, `4:3`, `3:4`, `21:9`, `9:21`
* **Resolution**: `480p` or `720p`
* **Seed**: optional integer for reproducible outputs
